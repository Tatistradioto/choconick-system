Para o cabeçalho do contrato PDF, coloque a imagem do timbrado aqui:

  public/timbrado-choconick.jpg

Copie o arquivo TIMBRADO_CHOCONICK.jpg para esta pasta com o nome timbrado-choconick.jpg
(nome em minúsculo). Exemplo no Windows (PowerShell):
  Copy-Item "C:\caminho\TIMBRADO_CHOCONICK.jpg" -Destination "public\timbrado-choconick.jpg"

O cabeçalho é exibido com 210mm × 52mm no topo de cada página. O conteúdo do contrato
começa em 58mm (abaixo do timbrado).

Sem este arquivo, o PDF será gerado com um cabeçalho simples em cor sólida.
